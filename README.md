# Stylish QR Code
![logo](icons/favicon.ico)
A professional, feature-rich web application for generating highly customizable and branded QR codes. Built using the our most  powerful `stylishQrGen` library, this tool allows users to go beyond standard black-and-white pixels to create unique, high-quality QR codes suitable for marketing, personal branding, and business use.

---

## 📄 Overview

**Stylish QR Code** provides a seamless interface for creating QR codes that stand out. Whether you need a Cyberpunk-themed code for a tech event, an Elegant gold-gradient code for a luxury brand, or a standard social media link with an integrated logo, this tool handles it automatically. It features smart auto-styling logic that adjusts colors and logos based on the data type you select, ensuring a professional look with zero design effort.

---

## ✨ Features

### 1. Advanced Styling
*   **Unique Shapes:** Choose from various dot styles (Square, Dots, Rounded, Extra-Rounded, Classy, etc.).
*   **Custom Corners:** Independently style the "Corner Squares" and "Corner Dots" with distinct shapes.
*   **Color & Gradients:** Apply single colors or complex gradients (Linear/Radial) with customizable rotation to the background, dots, and corners.
*   **Transparency:** Toggle transparent backgrounds for professional PNG overlays.

### 2. Branding & Logos
*   **Style Templates:** Instant presets like *Cyberpunk, Elegant, Modern, Funky, and Soft Style* that auto-load matched color palettes and custom themed logos.
*   **Social Presets:** Quick-access logos for Facebook, X (Twitter), Instagram, WhatsApp, YouTube, and more.
*   **Custom Uploads:** Upload your own brand logo and adjust its scale and margin within the QR code.

### 3. Comprehensive Data Templates
Generate specialized QR codes for:
*   **Communication:** WhatsApp, Telegram, SMS, Email, Phone, and FaceTime.
*   **Utility:** Wi-Fi networks (SSID/Password), vCard (Contact cards), and Geo-locations.
*   **Web & Social:** Standard URLs, Instagram profiles, Twitter accounts, and YouTube channels.
*   **Business:** Crypto payments (Bitcoin, Ethereum, etc.) and Calendar Events.

### 4. Professional Export Options
*   **High-Resolution Downloads:** Export in formats including **PNG, JPEG, WEBP, and SVG (Vector)**.
*   **Adjustable Quality:** Choose export sizes from 300px up to Ultra HD 2048px.
*   **Configuration Portability:** Export your entire design configuration as a **JSON** file and import it later to make quick edits.

## 🚀 Usage

### Running Locally
Since this is a client-side web application, no complex installation is required:

---

### Creating a QR Code
1.  **Select Style:** Click a **Quick Style** button to apply a baseline design.
2.  **Input Data:** Choose your **Data Type** (e.g., Wi-Fi) and fill in the required fields. The logo will automatically update to match the data type.
3.  **Fine Tune:** Open the **Dots**, **Corners**, or **Background** panels to customize colors and shapes.
4.  **Logo Adjustment:** Use the **Image Options** to resize or clear the center logo.
5.  **Download:** Select your preferred format and size, then click **Download QR Code**.
